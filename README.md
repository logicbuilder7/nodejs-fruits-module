# Node.js Fruits Module 🍎🍌🍊

A simple Node.js project demonstrating the use of the **module system** by organizing different fruits into separate files and importing them into a central file.

---

## 🎯 Project Purpose

This project was created to understand:

* How modular programming works in Node.js
* How to split code into reusable components
* How to use `require` and `module.exports` effectively

---

## 🛠️ Tech Stack

* Node.js
* JavaScript (CommonJS Modules)

---

## ⚙️ How to Run

```bash
node index.js
```

---

## ✨ Features

* Clean modular structure
* Separation of concerns using different files
* Easy to scale and extend with more modules

---
